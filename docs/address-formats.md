# 27 国地址格式规范

## 通用约定

- **真实组件**:来自 Overture/OSM 开放数据,有来源证据 —— 道路、行政区(区县/城市/州省)、邮编、坐标。
- **合成组件**:随机生成、明确标记 `synthetic` —— 室内层级(楼栋/单元/房间/APT 号);**仅中国**额外随机门牌号与小区名(已决策②)。其余国家门牌号一律真实(数据源自带)。
- **三语输出**:原文(native)/ English / 简体中文;门牌号、邮编等数字跨语言不变。
- **模板记号**:`%N`收件人 `%O`楼宇/小区 `%A`街道行(门牌+路+单元) `%D`区县 `%C`城市 `%S`州省 `%Z`邮编,`%n`换行。
- **表格读法**:每国一张表,行 = 字段(从最小的街道到最大的行政区,最后一行"完整地址"),列 = 原文 / English / 中文 / 来源(真实 or 随机)。
- **"从简到全"阶梯**:每国表下给出①街道行 → ②+城市 → ③+州省/邮编 → ④完整地址 的递进示例。

- **地图链接**:每条结果提供"坐标预览"与"骨架地址搜索"双链接(中国另有高德链接,坐标经 WGS→GCJ 转换);合成组件(中国门牌/小区、各国室内单元)不参与地图搜索,地图只对真实骨架负责。
- **邮编清洗**:源数据中不符合国家格式的邮编(电话号/城市名等脏值)在读取层清洗后由目录回填;显示的邮编要么合法要么为空。

> genaddr.com 逐国对齐结论见文末附录;该站仅 21 国、扁平四字段,本项目字段更完整、且提供三语与来源证据。
